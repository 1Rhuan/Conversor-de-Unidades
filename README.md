### Nosso tema:

**Conversor de Unidades (Múltiplo)**

Aplicação que converte valores entre diferentes unidades em pelo menos três
categorias (ex: Temperatura: Celsius/Fahrenheit; Distância: Km/Milhas; Peso: Kg/Libras). ​

Funcionalidades Mínimas: 
- Seleção da categoria de conversão. ​
- Entrada de valor e unidade de origem.​
- Exibição do resultado na unidade de destino.

# Como rodar o projeto — Backend, Testes e Frontend

Passos para executar o backend (Node.js), rodar os testes e executar o frontend.

## Pré-requisitos

- Node.js v20+.
- npm.
- Git.

## 1) Obter o código

```bash
git clone https://github.com/1Rhuan/Conversor-de-Unidades.git
cd Conversor-de-Unidades
```

## 2) Instalar dependências

```bash
cd backend
npm install
cd ..
```
---

## 4) Rodar o backend

2. Rodar:
```bash
cd backend/
node index.js
```
---

## 5) Rodar o frontend (local, sem servidor)

- Abra `frontend/index.html`
  ou
- Copie o caminho do arquivo e cole no nagevador.

## 6) Executar testes automatizados

- Testes com Jest:
```bash
npm test
```
